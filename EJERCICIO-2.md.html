# EJERCICIO 2

INICIAR Programa

CLASE Equipo
VARIABLES privadas: nombre, tipo, capacidad

    CONSTRUCTOR(nombre, tipo, capacidad)
        INICIALIZAR nombre, tipo, capacidad

    METODO toString()
        RETORNAR cadena con nombre, tipo, capacidad

    METODO obtenerNombre()
        RETORNAR nombre

    METODO obtenerCapacidad()
        RETORNAR capacidad

FIN CLASE

CLASE Red
VARIABLE privada: lista de equipos

    CONSTRUCTOR()
        INICIALIZAR lista de equipos

    METODO registrarEquipo(equipo)
        AÑADIR equipo a lista
        IMPRIMIR equipo registrado

    METODO listarEquipos()
        SI lista está vacía
            IMPRIMIR "La red no tiene equipos."
        SINO
            PARA CADA equipo EN lista
                IMPRIMIR equipo

    METODO simularCarga()
        SI lista está vacía
            IMPRIMIR "No hay equipos para simular carga."
        SINO
            PARA CADA equipo EN lista
                GENERAR carga aleatoria
                IMPRIMIR carga y verificar capacidad

FIN CLASE

CLASE SimuladorDeRed
METODO main()
CREAR instancia de Red
REPETIR
SOLICITAR al usuario elegir una acción
SEGUN acción elegida
CASO 1: Registrar equipo
CASO 2: Listar equipos
CASO 3: Simular carga
CASO 4: Salir
HASTA que acción sea Salir
FIN CLASE

FIN Programa
